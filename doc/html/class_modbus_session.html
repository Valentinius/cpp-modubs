void ModbusSession::addClient ( std::shared_ptr< ModbusClient >  modbus_client)

Метод для добавление MODBUS клиента явно. Причем клиентами могут быть все те объекты, классы которых являются наследниками абстрактного класса ModbusClient. Это позволяет в случае чего работать сразу с TCP/IP, а также, например, RS-232

Аргументы
modbus_client- указатель на клиент
Возвращает
индекс добавленного клиента в векторе клиентов сессии. Он необходим для того, чтобы в дальнейшем обращаться к данному клиенту через сессию.

◆ stop()

void ModbusSession::stop ( )

Метод для остановки сессии. До остановки должны заверщиться все запросы, которые было в сессии до того, как был вызван этот метод. Запросы, добавленные в сессию после вызова этого метода и до завершения сессии исполнены не будут
